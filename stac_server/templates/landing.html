STAC API

v1.0.0 + Qubed extensions

A fully compliant STAC API v1.0.0 is available at /api/stac/v1. Beyond the standard, Qubed adds hierarchical catalogue browsing: any collection can be drilled into key-by-key (e.g. class → stream → type → …) using the /catalog/… sub-tree, making it trivial to navigate massive multidimensional datasets without enumerating all items upfront.
